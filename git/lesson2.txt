git checkout - отмена изменений которые еще не закоммитили. Безвозвратно

git restore --staged - отмена если по ошибке сделали git add

git commit --amend - последующий коммит перезаписывает предыдущий дополняя новым / замена предыдущего коммита

git reset HEAD˜N - откатиться на N коммитов назад (тильда N)
