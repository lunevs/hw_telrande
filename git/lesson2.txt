git checkout - отмена изменений которые еще не закоммитили. Безвозвратно

git restore --staged - отмена если по ошибке сделали git add
git restore - восстановить файл из локального репозитория, например, если случайно удалил

git commit --amend - последующий коммит перезаписывает предыдущий дополняя новым / замена предыдущего коммита

git reset HEAD˜N - откатиться на N коммитов назад (тильда N)

git revert HEAD - создает обратный коммит к нашему последнему. Хорошо помогает если надо исправить последний запушенный коммит

git checkout -b BranchName - создание ветки и переключение на нее

git checkout BranchName - переключение на ветку

git merge BranchName - слияние ветки с main
