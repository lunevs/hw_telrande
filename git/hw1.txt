1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Можно воспользоваться командой git clone и указать ссылку на репозиторий


2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

git init - для инициализации репозитория
git add . - добавляем локальную папку или файл в локальный репозиторий git (staging area), они теперь готовы к коммиту
git commit -m "описание первого коммита" - отправляем снэпшот изменений в локальный репозиторий, они теперь готовы к отправке на сервер
git remote add origin "адрес удаленного репозитория" - добавляем адрес удаленного репозитория
git push - отправляем на сервер

тут будет еще ньюанс с авторизацией. Нужно создать сертификаты локально и загрузить публичный ключ на гитхаб и потом уже можно будет делать push


3. Как получить на локальный компьютер изменения с github.com

git pull origin Branch_Name - выгружаем обновление и указываем какая именно ветка нас интересует. Pull - не только выгружает, но и сразу вызывает merge. 

либо второй вариант:

git fetch - выгружает репозиторий
git merge - слияние выгруженных обновленных данных с локальными



4. Какую информацию нам дают команда :
git status - показывает статус файлов в локальном репозитории и рабочей папке 
git log - показывает историю commit в данной папке
git remote -v  - показывает список адресов удаленных репозиториев


Ответы на вопросы 1-3 должны содержать конкретные команды git
